산정 기준 (별표 5 2.1.3)KDS 31 30 15
① 고가수조 양수펌프 Q
Q = 시간최대 예상급수량 [ℓ/hr]
→ LPM 환산: ÷ 60
① 자동부스터 펌프 Q
전체 Q = 순간최대 예상급수량 [ℓ/min]
각 펌프 Q = 전체 Q ÷ 운전대수 [LPM]
※ 펌프 2대 이상 병렬 설치 필수
② 양정 H [m]
H = hₐ + hpv + hₒ + 안전율
hₐ : 실양정 (흡입~토출 높이차)
hpv: 배관마찰손실 (L×R + L×R×0.5)
hₒ : 말단기구 최저필요수압
③ P [HP]
P = Q×H×(1+여유율%) ÷ (4500×η)
η = 0.55 기준 / 여유율 10~15% 권장
④ 감압밸브 필수 조건 (2.1.7)
말단기구 수압 ≥ 550 kPa(56.1m) 시
감압밸브(PRV) 또는 급수 조닝 필수
⑤ 위생기구 최저필요수압
욕조/세면기: 55kPa(5.6m)
샤워기: 70kPa(7.1m)
소변기·대변기(밸브): 100kPa(10.2m)
압력식 샤워기: 130kPa(13.3m)

상단 배너 [✏️ 정보 수정] 또는 [프로젝트 정보] 탭에서 공사명·작성자·날짜를 입력합니다.

STEP 2 — 8종 펌프 순서 산정
상단 진행 상태 바에서 8개 펌프를 확인하며 순서대로 산정합니다.
각 탭의 서브탭(난방/냉방, PIT/우수, 순환/재순환, 고가/부스터)을 전환하며 결과가 독립 저장됩니다.
3
⚡
STEP 3 — 자동 산정 및 검증
입력값 입력 시 실시간 자동 계산됩니다. [산정] 버튼으로 수동 갱신도 가능합니다.
여유율 초과, 집수조 30분 검증, 550kPa 초과 등 경고가 자동 표시됩니다.

📄
STEP 4 — A4 미리보기 및 출력

8종 모두 작성하면 전체 산출서 + EPI 장비선정계산서가 완성됩니다.

냉난방 순환펌프 (별표 1) — hₐ=0 밀폐회로
Q = 부하[kcal/hr] × (1+여유율) ÷ ΔT[℃] ÷ 60 → LPM / 난방ΔT=10℃, 냉방ΔT=5℃
H 소계 = L×R + L×R×0.5 + 코일저항 + 보일러(냉동기)저항 (실양정 hₐ=0, 팽창탱크 폐회로)
전양정 H = H소계 + H소계×안전율% / P = Q×H×(1+여유율%) ÷ (4500×η)
🚰
오·배수 펌프 (별표 6) — 200% 기준
PIT Q = 시간최대 유입량×200% / 30분 이내 완전 배출 필수
우수 Q = (mm×m²)÷60÷대수 / H = 층고+L×R+L×R×0.5+안전율
집수정 Ø450 이상 / D600 이상 / 고형물: 오수Ø50, 잡배수Ø25
🔥
급탕 순환/재순환 펌프 (별표 5)
순환 Q = 시간최대급탕량[ℓ/hr] ÷ 2 ÷ 60 → LPM
재순환 Q = 시간최대급탕량[ℓ/hr] ÷ (출탕℃ - 냉각허용℃) → LPM (출탕60℃, 냉각5℃ 기준)
H = R[mmAq/m] × I[m] × (1+K) ÷ 1000 → m / η = 0.35
🚿
급수 펌프 (별표 5 2.1.3)
고가수조 Q = 시간최대급수량÷60 / 자동부스터 Q = 순간최대÷운전대수
H = hₐ+hpv+hₒ+안전율 / η = 0.55 / ≥550kPa → 감압밸브 필수

공통 — 동력 선정 및 여유율
P(HP) 계산값의 직상위 표준HP 선택 / HP≤¼ → 1Φ 220V / HP>¼ → 3Φ 380V
전동기 여유율 10~15% 권장 / 20% 초과 금지 (에너지 절약 설계 기준)

냉난방 설계 팁
단위마찰손실 R = 0.02~0.04 mAq/m / 보일러저항 5~8m / 냉동기코일 3~5m
판형 열교환기 차압 70kPa 이하 유지 / 인버터 제어로 EPI 가점 확보
🚰
오배수 설계 팁
집수조 유효용량 = Q×5~10분 이상 확보 / 2대 교번운전 필수
우수펌프: 지역 기상청 10년 빈도 강수량 적용 / 우수 η=0.30 적용
🔥
급탕 설계 팁
중규모: R=15~20 mmAq/m, K=1.0~1.2 / 소규모: R=10~20, K=1.0~1.5
재순환펌프 Q = 시간최대급탕량[ℓ/hr] ÷ (출탕-냉각허용)[℃] — 단위 주의
예비 1대 필수 / 급탕 출탕 60℃, 냉각허용 5℃ 기준
